GamepadBT-Server by BeardoGuy for Symbian S60 Client devices running GamepadBT app. 08/07/2023
https://github.com/BeardoGuy/GamepadBT-Server/

Keymap:
0-9 number keys on phone correspond to 0-9 keystrokes on PC
Arrows (Joypad) = Arrow keys on Keyboard
OK key = 'Enter'
* = '*'
# = '/'
Left SoftKey = '+'
RightSoftKey = '-'
Green Call Key (Yes key) = '.'
Clear key "C" = 'BackSpace'
